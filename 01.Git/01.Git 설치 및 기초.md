# Git 설치 및 기초

## Git 설치

> windows 사용자 기준

### winget 을 통해 git 설치

```bash
winget install -e --id Git.Git
```

### `git init`

폴더 또는 디렉토리를 git 저장소로 초기화

```bash
git init
```

### `git add`

새로 추가된 파일, 변경된 파일, 삭제된 파일에 대한 변경사항을 git에 추가

```bash
git add -A # 변경된 파일 전체를 git 변경사항에 추가

git add <파일경로> # 특정한 파일을 선택하여 git 변경사항에 추가
```

### `git commit`

git 변경사항에 추가된 파일을 로컬 저장소에 기록

```bash
git commit -m "<커밋 메세지>" # 커밋 메세지를 추가하여 변경사항을 커밋
```

### `git push`

로컬 저장소에 기록된 변경사항을 원격저장소에 푸쉬

```bash
git push
```
